# 🖨️ PDF_TEMPLATE_SPEC.md — InvoicePK PDF Template Specification

> This document defines the design, layout, and implementation of the invoice PDF.
> Frontend developer owns `InvoicePDFTemplate.tsx` — the visual component.
> Backend developer owns `GET /api/invoices/:id/pdf` — the route that renders and streams it.
> Both must agree on the props interface before either starts building.

---

## Ownership Split

| Part | Owner | File |
|---|---|---|
| PDF visual template | Frontend developer | `components/InvoicePDF/InvoicePDFTemplate.tsx` |
| PDF API route (stream) | Backend developer | `app/api/invoices/[id]/pdf/route.ts` |
| PDF save route (Cloudinary) | Backend developer | `app/api/invoices/[id]/pdf/save/route.ts` |
| PDF download button UI | Frontend developer | `components/InvoicePDF/DownloadButton.tsx` |

---

## Props Interface — Agree Before Building

This is the data shape the backend passes into `<InvoicePDFTemplate />`. Neither developer changes this without notifying the other.

```typescript
// types/index.ts — already defined, shown here for reference
export interface PDFInvoiceProps {
  invoice: {
    invoiceNumber: string;
    status:        string;
    issueDate:     string;   // formatted: "01 June 2026"
    dueDate?:      string;   // formatted: "30 June 2026"
    currency:      'PKR' | 'USD';
    gstType:       'standard' | 'zero_rated' | 'exempt';
    gstRate:       number;
    subtotal:      number;
    gstAmount:     number;
    total:         number;
    whtApplicable: boolean;
    whtRate:       number;
    whtAmount:     number;
    netPayable:    number;
    notes?:        string;
    items: Array<{
      description: string;
      quantity:    number;
      unitPrice:   number;
      amount:      number;
      sortOrder:   number;
    }>;
  };
  business: {
    name:     string;
    ntn?:     string;
    strn?:    string;
    address?: string;
    logoUrl?: string;
  };
  client?: {
    name:     string;
    email?:   string;
    address?: string;
    ntn?:     string;
  };
}
```

## Design Tokens

Use these exact values in `@react-pdf/renderer` StyleSheet. Do NOT reference `styles/theme.ts` here — `@react-pdf/renderer` runs in a Node.js PDF context, not in the browser. Hardcode these values in the PDF component only.

```typescript
const colors = {
  primary:      '#1e3a5f',   // Navy blue — header bar, table header bg
  primaryLight: '#d5e8f0',   // Light blue — alternating table rows
  accent:       '#198754',   // Green — Net Payable box border + label
  text:         '#1a1a1a',   // Near black — all body text
  textMuted:    '#555555',   // Gray — secondary labels
  border:       '#cccccc',   // Light gray — table borders, dividers
  white:        '#ffffff',
  netPayableBg: '#f0faf4',   // Very light green — Net Payable box bg
};

const fonts = {
  // @react-pdf/renderer uses built-in fonts
  // Use 'Helvetica' for body text, 'Helvetica-Bold' for headings
  body:   'Helvetica',
  bold:   'Helvetica-Bold',
};

const spacing = {
  page:      30,   // page padding (pt)
  section:   16,   // between major sections
  small:     8,    // between small elements
  micro:     4,
};
```

```typescript

import {
  Document, Page, View, Text, Image, StyleSheet
} from '@react-pdf/renderer';
import { PDFInvoiceProps } from '@/types';
import { formatPKR, formatUSD } from '@/lib/gst';

const styles = StyleSheet.create({
  page: {
    fontFamily:  'Helvetica',
    fontSize:    10,
    color:       '#1a1a1a',

    paddingBottom: 30,

  },

  },

  },

    color:      '#1e3a5f',
    marginBottom: 4,
  },

    fontSize: 9,

  },

    flexDirection:   'row',
    justifyContent:  'space-between',
    backgroundColor: '#1e3a5f',
    padding:         10,
    borderRadius:    4,
    marginBottom:    16,
  },
  detailLabel: {
    fontSize:   8,

    marginBottom: 2,
  },
  detailValue: {
    fontSize:   11,
    fontFamily: 'Helvetica-Bold',
    color:      '#ffffff',
  },

  // ── Billed To ────────────────────────────────────
  billedToSection: {
    marginBottom: 16,
  },
  sectionLabel: {
    fontSize:     8,
    fontFamily:   'Helvetica-Bold',

    letterSpacing: 1,
    marginBottom:  4,
  },
  clientName: {
    fontSize:   12,
    fontFamily: 'Helvetica-Bold',
    marginBottom: 2,
  },
  clientDetail: {
    fontSize:     9,
    color:        '#555555',
    marginBottom: 2,
  },

  // ── Line Items Table ──────────────────────────────
  table: {
    marginBottom: 16,
  },
  tableHeader: {
    flexDirection:   'row',
    backgroundColor: '#1e3a5f',
    padding:         8,
  },
  tableHeaderText: {
    fontSize:   9,
    fontFamily: 'Helvetica-Bold',

export function InvoicePDFTemplate({ invoice, business, client }: PDFInvoiceProps) {
  const fmt = invoice.currency === 'PKR' ? formatPKR : formatUSD;
  const currencyLabel = invoice.currency;

  return (
    <Document>
      <Page size="A4" style={styles.page}>

        {/* ── HEADER ──────────────────────────────── */}
        <View style={styles.header}>
          {business.logoUrl ? (
            <Image src={business.logoUrl} style={styles.logo} />
          ) : (
            <View style={{ width: 80 }} />
          )}
          <View style={styles.businessInfo}>
            <Text style={styles.businessName}>{business.name}</Text>
            {business.ntn && (
              <Text style={styles.businessDetail}>NTN: {business.ntn}</Text>
            )}
            {business.strn && invoice.gstType === 'standard' && (
              <Text style={styles.businessDetail}>STRN: {business.strn}</Text>
            )}
            {business.address && (
              <Text style={styles.businessDetail}>{business.address}</Text>
            )}
          </View>
        </View>

        {/* ── INVOICE DETAILS BAR ─────────────────── */}
        <View style={styles.detailsBar}>
          <View>
            <Text style={styles.detailLabel}>INVOICE NO.</Text>
            <Text style={styles.detailValue}>{invoice.invoiceNumber}</Text>
          </View>
          <View>
            <Text style={styles.detailLabel}>ISSUE DATE</Text>
            <Text style={styles.detailValue}>{invoice.issueDate}</Text>
          </View>
          {invoice.dueDate && (
            <View>
              <Text style={styles.detailLabel}>DUE DATE</Text>
              <Text style={styles.detailValue}>{invoice.dueDate}</Text>
            </View>
          )}
          <View>
            <Text style={styles.detailLabel}>CURRENCY</Text>
            <Text style={styles.detailValue}>{currencyLabel}</Text>
          </View>
        </View>

        {/* ── BILLED TO ───────────────────────────── */}
        {client && (
          <View style={styles.billedToSection}>
            <Text style={styles.sectionLabel}>Billed To</Text>
            <Text style={styles.clientName}>{client.name}</Text>
            {client.address && (
              <Text style={styles.clientDetail}>{client.address}</Text>
            )}
            {client.ntn && invoice.whtApplicable && (
              <Text style={styles.clientDetail}>NTN: {client.ntn}</Text>
            )}
            {client.email && (
              <Text style={styles.clientDetail}>{client.email}</Text>
            )}
          </View>
        )}

        {/* ── LINE ITEMS TABLE ────────────────────── */}
        <View style={styles.table}>
          {/* Table Header */}
          <View style={styles.tableHeader}>
            <Text style={[styles.tableHeaderText, styles.colDescription]}>Description</Text>
            <Text style={[styles.tableHeaderText, styles.colQty]}>Qty</Text>
            <Text style={[styles.tableHeaderText, styles.colUnitPrice]}>Unit Price</Text>
            <Text style={[styles.tableHeaderText, styles.colAmount]}>Amount</Text>
          </View>
          {/* Table Rows */}
          {invoice.items
            .sort((a, b) => a.sortOrder - b.sortOrder)
            .map((item, index) => (
              <View
                key={index}
                style={[styles.tableRow, index % 2 === 1 ? styles.tableRowAlt : {}]}
              >
                <Text style={[styles.tableCell, styles.colDescription]}>{item.description}</Text>
                <Text style={[styles.tableCell, styles.colQty]}>{item.quantity}</Text>
                <Text style={[styles.tableCell, styles.colUnitPrice]}>{fmt(item.unitPrice)}</Text>
                <Text style={[styles.tableCell, styles.colAmount]}>{fmt(item.amount)}</Text>
              </View>
            ))
          }
        </View>

        {/* ── TOTALS ──────────────────────────────── */}
        <View style={styles.totalsSection}>
          <View style={styles.totalsRow}>
            <Text style={styles.totalsLabel}>Subtotal</Text>
            <Text style={styles.totalsValue}>{fmt(invoice.subtotal)}</Text>
          </View>

          {invoice.gstType === 'standard' && (
            <View style={styles.totalsRow}>
              <Text style={styles.totalsLabel}>GST ({invoice.gstRate}%)</Text>
              <Text style={styles.totalsValue}>{fmt(invoice.gstAmount)}</Text>
            </View>
          )}

          {invoice.gstType === 'zero_rated' && (
            <View style={styles.totalsRow}>
              <Text style={styles.totalsLabel}>GST (Zero-Rated — IT Export)</Text>
              <Text style={styles.totalsValue}>{fmt(0)}</Text>
            </View>
          )}

          <View style={styles.dividerLine} />

          <View style={styles.totalsRow}>
            <Text style={[styles.totalsLabel, { fontFamily: 'Helvetica-Bold' }]}>Total</Text>
            <Text style={styles.totalsValue}>{fmt(invoice.total)}</Text>
          </View>

          {invoice.whtApplicable && (
            <View style={styles.totalsRow}>
              <Text style={styles.totalsLabel}>WHT ({invoice.whtRate}%) Deduction</Text>
              <Text style={[styles.totalsValue, { color: '#c62828' }]}>- {fmt(invoice.whtAmount)}</Text>
            </View>
          )}

          <View style={styles.netPayableBox}>
            <Text style={styles.netPayableLabel}>NET PAYABLE</Text>
            <Text style={styles.netPayableValue}>{fmt(invoice.netPayable)}</Text>
          </View>
        </View>

        {/* ── NOTES ───────────────────────────────── */}
        {invoice.notes && (
          <View style={styles.notesSection}>
            <Text style={[styles.sectionLabel, { marginBottom: 4 }]}>Notes</Text>
            <Text style={styles.notesText}>{invoice.notes}</Text>
          </View>
        )}

        {/* ── FOOTER ──────────────────────────────── */}
        <View style={styles.footer} fixed>
          <Text style={styles.footerText}>Thank you for your business.</Text>
          <Text style={styles.footerText}>Generated by InvoicePK</Text>
        </View>

      </Page>
    </Document>
  );
}
```

## Backend: How to Render and Stream the PDF

```typescript
// app/api/invoices/[id]/pdf/route.ts
import { renderToBuffer } from '@react-pdf/renderer';
import { InvoicePDFTemplate } from '@/components/InvoicePDF/InvoicePDFTemplate';

// Inside the GET handler, after fetching invoice from MongoDB:
const pdfBuffer = await renderToBuffer(
  <InvoicePDFTemplate
    invoice={invoiceData}
    business={businessData}
    client={clientData}
  />
);

const filename = `${invoice.invoiceNumber}_${client?.name?.replace(/\s+/g, '_') ?? 'Invoice'}.pdf`;

return new NextResponse(pdfBuffer, {
  status: 200,
  headers: {
    'Content-Type':        'application/pdf',
    'Content-Disposition': `attachment; filename="${filename}"`,
    'Content-Length':      pdfBuffer.length.toString(),
  },
});
```

## Conditional Display Rules

| Condition | What to show / hide |
|---|---|
| `business.logoUrl` is null | Hide logo space entirely — right-align business info |
| `business.strn` is null | Hide STRN line from header |
| `invoice.gstType === 'exempt'` | Hide GST row from totals entirely |
| `invoice.gstType === 'zero_rated'` | Show GST row with "Zero-Rated — IT Export" label and PKR 0 |
| `invoice.whtApplicable === false` | Hide WHT row from totals |
| `client` is null | Hide "Billed To" section entirely |
| `client.ntn` is null | Hide client NTN line |
| `invoice.dueDate` is null | Hide Due Date from details bar |
| `invoice.notes` is null | Hide Notes section entirely |

## Install

```bash
npm install @react-pdf/renderer
npm install --save-dev @types/react-pdf
```

> ⚠️ `@react-pdf/renderer` must only be used server-side in API routes. Do not import it in Client Components — it will crash the browser bundle. If you need a PDF preview in the UI, use an `<iframe>` pointing to the `/api/invoices/:id/pdf` route.
